## What is ExamineX?

ExamineX is a modern upgrade for your Umbraco website's search capabilities. Instead of managing search files on your own servers, ExamineX connects your website to professional cloud search services like [Azure AI Search](https://azure.microsoft.com/en-au/services/search/) or [Elastic Search](https://www.elastic.co/).
